**SUPERSTORE SALES ANALYSIS**

This is a project made using MySQL and Power BI.
It describes the information about customer, product and sales of the store chain across the United States.

*The project was made by the following steps :*
1) Data Collection
2) Data Cleaning
3) Feature Engineering
4) Exploratory Data Analysis (EDA)
5) Data Visualization using Power BI
